# Taxi App

A simple Taxi App with **signup, login, and home pages**.  
The app uses **Node.js** for the backend and stores user data in `users.json`.

---

## Features

- Signup with: Full Name, Username, Password, Phone, Gender  
- Login with Username + Password  
- Home page after successful login  
- Login-failed page if credentials are wrong  
- “Remember me” functionality in login form  
- Frontend built with HTML, CSS, and JS  

---

## How to Run Locally

1. Clone the repo:
git clone https://github.com/YOUR_USERNAME/taxi-app.git
cd taxi-app

2. npm install
3. node server.js
4- http://localhost:3000/index.html
